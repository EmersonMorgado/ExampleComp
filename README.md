# Cálculo de Salário Líquido

Faça um programa que leia os dados de um funcionário (**nome** e **salário bruto**) e depois mostre o **salário líquido**, que consiste no valor bruto descontado **impostos** e **previdência**.

## Regras

1. **Imposto:** 20%  
2. **Previdência:** 10%

## Exemplo

Nome: José
Salário bruto: 4000.00

**Retorno**
Salário líquido = 2800.00
